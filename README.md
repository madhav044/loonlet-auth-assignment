# Loonlet Auth Assignment

A Django-based authentication system built for the Loonlet assignment.

## Features

- User Registration
- User Login
- JWT Authentication
- Protected API Endpoints
- Secure User Management

## Tech Stack

- Python
- Django
- Django REST Framework
- JWT Authentication

## Installation

pip install -r requirements.txt

## Run the Project

python manage.py migrate
python manage.py runserver

The application will be available at:

http://127.0.0.1:8000/
